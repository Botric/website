Windows issues

  • Error code: 0xC1900101

      • This is the most common issue with windows installation is that you can pass through the main sections but will fail on the drive selections or on the drive updates

    • resolution: unplug all storage that windows isnt being installed on
  • Error code: User account

      • when installing windows and running through the windows setup process you need to create an account if you have a windows 10/11 home licence you will need to sign in with an outlook account

    • resolution: make sure the machine is never connected to the internet and you will be able to create a local account
